Instructions:

Hello World! This program is made to modify / generate web pages that have been designed by Alice Easter

To start pick a type of page from the drop down menu

From there, if you want to create a new template click parse html, else if you have html to modify copy it into the text area, then press parse

Modify the content as desired, then when all is done and all fields filled out, click the build html button on the bottom of the screen

From here a text area will appear, click in it, press CMD-A on a Mac or CTRL-A on a PC to select all, then copy

You can paste that HTML into Ingeniux and then include the client side script component that is appropriate for the situation

Note: For each page you must insert appropriate component from '/Internal Components/Client Side Script Components/Alice'
